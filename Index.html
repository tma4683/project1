Get ready to soar with Jump Rush!

JumpRush is a simple game involving an astronaut that must jump to avoid obstacles in order to achieve the highest score.
Developed by five web developers using Javascript, CSS, and HTML.

Rules Of The Game →

Rule 1: Press Spacebar to jump.
Rule 2: Avoid obstacles.
Rule 3: Try to beat your high score.
